# 识字消消乐

一款面向儿童的汉字学习消消乐游戏。

## 功能特点

- 🎮 经典三消玩法，简单易上手
- 📚 10个主题关卡，60个精选汉字
- 🔊 消除时播放汉字读音，强化记忆
- 🎨 可爱的卡通风格，吸引儿童注意力
- 📈 渐进式难度，适合不同年龄段

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

## 音频资源获取

游戏需要汉字读音音频文件。推荐以下方式获取：

### 方式一：百度智能云语音合成（推荐）

1. 访问 [百度智能云](https://cloud.baidu.com/product/speech/tts)
2. 注册账号并开通语音合成服务
3. 使用 API 或控制台生成汉字读音
4. 将生成的 MP3 文件放入 `assets/audio/pinyin/` 目录

### 方式二：讯飞开放平台

1. 访问 [讯飞开放平台](https://www.xfyun.cn/services/online_tts)
2. 注册并开通在线语音合成
3. 生成汉字读音文件

### 方式三：使用现成的汉字读音资源

- [chinese-tts](https://github.com/jayin/chinese-tts) - 开源汉字语音库
- [pinyin-data](https://github.com/mozillazg/pinyin-data) - 拼音数据

### 音频文件命名

汉字读音文件应按拼音命名，例如：

- 一 → `yi.mp3`
- 二 → `er.mp3`
- 三 → `san.mp3`
- 天 → `tian.mp3`

## 游戏说明

1. 点击选择一个汉字格子
2. 点击相邻的格子进行交换
3. 三个相同汉字连成一线即可消除
4. 消除时会播放该汉字的读音
5. 达到目标数量即可过关

